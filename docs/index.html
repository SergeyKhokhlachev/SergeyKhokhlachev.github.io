  • Все UI компоненты выполены в виде классов, доступные в глобальной области видимости, как свойство обьекта app (window.app)
new app.Collapse(document.querySelector('.js-collapse'));
  • Экземпляры классов доступны по ссылке, как соответствующее свойство целевого DOM элемента
const myCollapse = document.querySelector('.js-collapse');
myCollapse.collapse.update();
